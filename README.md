# 🎓 Student Management System

🚀 A full-stack web application to manage student records efficiently using Java technologies.

---

## ✨ Features

✔️ Add New Student
✔️ View All Students
✔️ Edit Student Details
✔️ Delete Student
✔️ Clean and User-Friendly UI
✔️ MVC Architecture (Model-View-Controller)

---

## 🛠️ Tech Stack

* 💻 Java (JDK 8+)
* 🌐 JSP & Servlets
* 🗄️ MySQL
* 🔌 JDBC
* 🎨 HTML, CSS

## ⚙️ Setup Instructions

1️⃣ Clone the repository
2️⃣ Import project into Eclipse
3️⃣ Configure Apache Tomcat Server
4️⃣ Create MySQL database:

```sql
CREATE DATABASE studentdb;
```

5️⃣ Update DB credentials in `DBConnection.java`
6️⃣ Run the project on server
